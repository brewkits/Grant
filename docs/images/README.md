## How to Add Screenshots

1. Run the demo app on Android and iOS
2. Take screenshots of the permission flows
3. Use a screen recorder to create the demo GIF
4. Optimize images for web (max 500KB each)
5. Name files according to the list above
6. Update README.md with actual image links

## Image Guidelines

- Format: PNG for screenshots, GIF for animations
- Max size: 500KB per image
- Resolution: 1080p for Android, Retina for iOS
- Compress images before committing
